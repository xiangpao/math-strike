# Minecraft: 数字突击 - 全新重制版

## 项目综述
本项目是一个基于《我的世界》(Minecraft) 主题重制的弹幕射击与算数结合的网页游戏。在保留原本“算数打怪”核心趣味的基础上，加入了真实的硬核弹幕躲避机制和丰富的特效，带来极具挑战性的街机体验。

## 游戏特色
1. **双线控制机制**：玩家不仅需要计算屏幕上的数学题来射击敌人，还需要使用方向键（上/下/左/右）在全屏范围内灵活走位，躲避敌人的弹幕。
2. **丰富的敌军 AI**：
   - **普通僵尸**：匀速直线下降。
   - **骷髅射手**：半空悬停，发射抛射追踪子弹。
   - **苦力怕**：加速冲刺的自爆追踪者。
   - **末影人**：随机瞬移。
3. **史诗级 Boss 战**：每个关卡末尾都有强大的 Boss（如变异蜘蛛骑士、凋灵、末影龙、凋灵风暴），拥有满屏弹幕和高血量，需要连续答题才能击败。
4. **动态难度系统**：算术题难度随关卡线性提升（最大值 = 关卡数 × 10，最高封顶 100），并逐渐引入混合运算（如 a+b-c）以平滑难度曲线。答错不再打断连击，降低挫败感。
5. **强力道具池**：包含回复生命的金苹果、加速减速的迅捷药水、穿甲攻击的钻石剑以及提供无敌护盾的不死图腾。
6. **火力升级系统**：连击（Combo）达到 3 时升级为三向散弹，达到 5 时升级为五向粒子特效散弹。连击只有在受击或漏怪时才会归零。受伤时武器等级下降 1 级。
7. **复古视听体验**：内置 8-bit 音效合成器，配合动态像素星空背景和火焰粒子特效。

## 技术框架与实施路径
本项目采用纯前端技术栈进行开发。

### 核心技术栈
- **语言**: TypeScript / HTML5 / Vanilla CSS
- **构建工具**: Vite
- **DOM / Canvas 操作**: 游戏核心逻辑可能使用原生 Canvas API 或 DOM 节点进行渲染 and 动画处理。

### 本地运行指南
1. 确保已安装 Node.js 和 npm。
2. 克隆或下载本项目到本地。
3. 在项目根目录（包含 `package.json` 的目录）打开终端。
4. 运行 `npm install` 安装依赖。
5. 运行 `npm run dev` 启动本地开发服务器。
6. 默认情况下，应用将在 `http://localhost:5173` 或控制台提示的地址运行，在浏览器中打开即可体验。

### 部署与上线分享
该项目为纯前端静态页面，非常适合通过静态网站托管服务进行部署。

1. **打包构建**:
   在本地运行 `npm run build`，Vite 会将 TypeScript 代码编译并打包，最终生成的静态文件将放置在 `dist` 目录下。

2. **推荐部署方案**:
   - **EdgeOne Pages** (当前配置): 本项目已针对 EdgeOne Pages 进行深度适配，支持通过 Antigravity 自动化部署工具直接发布。
   - 其他方案: 也可将 `dist` 目录内容上传至 Vercel, Netlify 或 GitHub Pages。
